Installasjon og kjøring av Docker

Tidligere måtte var det kun Linux som hadde en funksjonell native Docker-implementasjon, men native docker for Mac og Windows har nå blitt såpass bra og stabilt at det er å foretrekke over å bruke virtualisert Docker via Docker Toolbox.

Docker

For å installere docker-core, gå til Docker Store, og velg versjonen for ditt OS fra listen. Følg så installasjonsinnstruksjonene.

Docker Machine

For å installere docker-machine, gå til Docker Docs, og følg installasjonsinstruksjonene. Merk at punkt 1, "Install the docker binary", allerede er utført når du har installert Docker i avnsittet over.
