Installasjon og kjøring av Docker

Tidligere var det kun Linux som hadde en funksjonell native Docker-implementasjon, og både Mac- og Windows-brukere måtte hoppe gjennom noen ekstra ringer. Native Docker for Mac og Windows har heldigvis nå blitt såpass bra og stabilt at det er å foretrekke over å bruke virtualisert Docker via Docker Toolbox.

Docker

For å installere docker-engine, gå til Docker Store, og velg versjonen for ditt OS fra listen. Følg så installasjonsinstruksjonene.

Docker Machine

For å installere docker-machine, gå til Docker Docs, og følg installasjonsinstruksjonene. Merk at punkt 1, "Install the docker binary", allerede er utført når du har installert Docker i avnsittet over.

Annen nødvendig programvare som forventes installert

  • Java 8 SDK. (Trenger ikke nødvendigvis være Oracle).
  • Maven 3
  • En fritt valgt SSH-client (OpenSSH, Putty eller lignende).
  • Programvare for filoverføring, som støtter SCP eller FTP.
