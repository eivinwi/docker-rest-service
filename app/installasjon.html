Installasjon og kjøring av Docker

Hvordan docker installeres er forskjellig avhengig av operativsystem. Installasjon og bruk av Docker er noe enklere og krever færre steg på Linux, men man kan også fint bruke Mac eller Windows. Lenker til de separate installasjonsinnstruksene finnes i tabellen under.

Operativsystem Informasjon
Linux Docker
Linux DockerMachine
Enkel, rett frem installasjon. Separat installasjon for Docker-machine.
Krever 64-bits versjon av Linux.
Ikke installer direkte fra package-managere som Apt/Yum/Pacman da docker konstant oppdateres, og versjonene derfra ofte blir utdaterte.
Virtualbox må også installeres for den siste delen av workshoppen.
Mac
Relativt enkel installasjon.
Krever minst OS X 10.8 Mountain Lion.
Krever Virtualbox (Dersom du har Virtualbox installert fra før huker du ikke av på installering av Virtualbox under installasjonen)
Windows
Litt flere steg enn på Linux/Mac, men selve installasjonen er relativt grei.
Krever minst Windows 7.
Krever 64-bits Windows.
Krever at Virtualization er aktivert i BIOS. (Win8+: kan sjekkes i Oppgavebehandling -> Ytelse)
Krever Virtualbox (Dersom du har Virtualbox installert fra før huker du ikke av på installering av Virtualbox under installasjonen)
Etter installasjon: for å kjøre Docker-kommandoer brukes nylig installerte Docker Quickstart Terminal.

Annen nødvendig programvare som forventes installert

  • Java 8 SDK. (Trenger ikke nødvendigvis væore Oracle).
  • Maven 3
  • En fritt valgt SSH-client (OpenSSH, Putty eller lignende).
  • Programvare for filoverføring, som støtter SCP eller FTP.
Merk: OpenSSH følger med i Docker Toolbox, støtter ssh/scp, og kan brukes fra Docker Quickstart Terminal!
