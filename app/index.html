Velkommen til Docker Workshop!

Målet med denne workshoppen er å gi en grunnleggende introduksjon til hva Docker er, hvordan det kan brukes, samt litt fordeler og ulemper med å bruke Docker i produksjon.

Etter den innlendende introduksjon om Docker blir det løsing av oppgaver, hovedsaklig knyttet til Docker Engine. Halvveis i oppgavene tar vi en pause med en prat om de andre verktøyene i verktøykassen til Docker, som Docker Machine og Docker Swarm, før de siste oppgavene skal løses.

Opplegg:

  • Intro
  • Installasjon av Docker
  • Oppgave 1 - Bygge Docker Images
  • Oppgave 2 - Modifisering av Dockerfile
  • Oppgave 3 - Dockerisering av Java-applikasjoner
  • Oppgave 4 - Skyer
  • Pause med en liten intro til Docker-Machine
  • Oppgave 5 - Docker Machine
  • Oppgave 6 - Docker Swarm
  • Oppgave 7 - Datavolumer og Logging
  • Oppgave 8 - Databaser
  • Avsluttende prat med oppsummering
