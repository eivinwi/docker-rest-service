Velkommen til Docker Workshop!

Målet med denne workshoppen er å gi en grunnleggende introduksjon til hva Docker er, hvordan det kan brukes, samt litt fordeler og ulemper med å bruke Docker i produksjon. Mye av fokus går på å skjønne de underliggende konseptene, heller enn å kjøre på med verktøy man ikke har grunnlag for å vite om er nyttige eller ei.

Etter den innlendende introduksjon om Docker blir det løsing av oppgaver, hovedsaklig knyttet til Docker Engine. Halvveis i oppgavene tar vi en pause med en prat om de andre verktøyene i verktøykassen til Docker, som Docker Machine og Docker Swarm, før de siste oppgavene skal løses.

Koden som trengs kan hentes på GitHub med kommandoen 'git clone https://github.com/eivinwi/docker-javaapplication.git'. Denne prosjektet inneholder en mappe for hver oppgave med de nødvendige filene, og også en liten Java-applikasjon som skal brukes i senere oppgaver. Om du ikke har Git tilgjengelig kan det lastes ned som en zip-fil her.

Opplegg:

  • Intro (blir vist, kan også aksesseres her: Introduksjon til Docker)
  • Installasjon av Docker
  • Oppgave 1 - Bygge Docker Images
  • Oppgave 2 - Modifisering av Dockerfile
  • Oppgave 3 - Dockerisering av Java-applikasjoner
  • Oppgave 4 - Skyer
  • Pause med en liten intro til Docker-Machine (blir vist, kan også aksesseres her: Docker-Machine)
  • Oppgave 5 - Docker Machine
  • Oppgave 6 - Docker Swarm
  • Oppgave 7 - Datavolumer og Logging
  • Oppgave 8 - Databaser
  • Avsluttende prat med oppsummering (blir vist, kan også aksesseres her: Docker-oppsummering)
Alle oppgavesider kommer med en samling nyttige lenker nederst. Disse skal være nok til å løse oppgavene. Det blir dog opp til hver enkelt deltaker om man ønsker å lese grundig for å lære, eller kun lete etter de nødvendige kommandoer for å løse oppgavene. Man kan se an tidsbruken underveis.
Blir man ikke ferdig, så er det likevel anbefalt å lese gjennom oppgavene man ikke rakk. Disse gir nyttig innblikk i hva som finnes "in the magical world of Docker", samt diverse ting man bør vite om og passe seg for!
